Java Project

This Java project was a digital movie library that uses various algorithms. It refers to the different movie streaming providers, gives some information about the movie and allows the customer to save this movie to his watch list. This project includes binary search, linear search, bubble sort and insertion sort. The data was stored in .txt files that are personally created when a new user registers there.

An admin login allows access to the administration, where the admin can post new movies and revise old ones.
